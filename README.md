# Diagramix
**Diagramix** — веб-сервис для поддержки системного анализа с модулем генерации диаграмм и моделей на основе текстового описания предметной области.
Проект разрабатывается в рамках выпускной квалификационной работы на тему:
**«Проектирование и разработка веб-сервиса для поддержки системного анализа с модулем ИИ для генерации диаграмм и моделей»**

## Основная идея
Сервис позволяет пользователю:
* зарегистрироваться и войти в систему;
* создать проект;
* ввести текстовое описание предметной области;
* выбрать тип диаграммы;
* сгенерировать результат;
* сохранить проект;
* просматривать список своих проектов и открывать их повторно.
В перспективе система предусматривает:
* расширение набора поддерживаемых диаграмм;
* развитие модуля искусственного интеллекта;
* административную панель;
* расширенное управление пользователями и проектами.

## Запуск проекта

### 1. Backend
Перейти в папку backend:
```bash 
cd backend
```
Создать и активировать виртуальное окружение:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
Установить зависимости:
```bash
pip install -r requirements.txt
```
Запустить сервер:
```bash
uvicorn app.main:app --reload
```
После запуска backend будет доступен по адресу:
```bash
http://127.0.0.1:8000
```
Swagger-документация:
```bash
http://127.0.0.1:8000/docs
```

### 2. Frontend
Перейти в папку frontend:
```bash
cd frontend
```
Установить зависимости:
```bash
npm install
```
Запустить dev-сервер:
```bash
npm run dev
```
После запуска frontend будет доступен по адресу:
```bash
http://localhost:5173
```
