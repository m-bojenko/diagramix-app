# Diagramix Frontend MVP

## Экраны

### 1. Вход
Маршрут: `/login`

Элементы:
- логотип Diagramix
- заголовок "Вход в Diagramix"
- поле Email
- поле Пароль
- кнопка "Войти"
- ссылка "Забыли пароль?"
- ссылка "Нет аккаунта? Зарегистрироваться"

Переходы:
- кнопка "Войти" -> `/`
- ссылка регистрации -> `/register`

---

### 2. Регистрация
Маршрут: `/register`

Элементы:
- логотип Diagramix
- заголовок "Регистрация в Diagramix"
- поле Имя
- поле Email
- поле Пароль
- поле Подтвердите пароль
- чекбокс согласия
- кнопка "Создать аккаунт"
- ссылка "Уже есть аккаунт? Войти"

Переходы:
- кнопка "Создать аккаунт" -> `/`
- ссылка входа -> `/login`

---

### 3. Генерация диаграммы
Маршрут: `/`

Элементы:
- логотип Diagramix
- иконка профиля
- поле "Название проекта"
- textarea "Описание предметной области"
- загрузка файла PDF/Word
- select "Тип диаграммы"
- кнопка "Сгенерировать"

Переходы:
- кнопка "Сгенерировать" -> `/result`
- иконка профиля -> `/profile`

---

### 4. Результат
Маршрут: `/result`

Элементы:
- логотип Diagramix
- иконка профиля
- название проекта
- тип диаграммы
- блок отображения диаграммы
- блок отображения кода
- кнопка "Сохранить"
- кнопка "Назад"
- кнопка "Сгенерировать заново"

Переходы:
- "Назад" -> `/`
- "Сохранить" -> сохраняет проект
- "Сгенерировать заново" -> повторная генерация
- иконка профиля -> `/profile`

---

### 5. Список проектов
Маршрут: `/projects`

Элементы:
- логотип Diagramix
- иконка профиля
- таблица/список проектов
- кнопка "Новый проект"
- кнопки "Открыть"

Переходы:
- "Новый проект" -> `/`
- "Открыть" -> `/result`
- иконка профиля -> `/profile`

---

### 6. Профиль пользователя
Маршрут: `/profile`

Элементы:
- логотип Diagramix
- заголовок "Профиль пользователя"
- поле Имя
- поле Email
- поле Новый пароль
- поле Подтвердите пароль
- информационный блок:
  - дата регистрации
  - количество проектов
- кнопка "Сохранить изменения"
- кнопка "Назад"
- кнопка "Выйти из аккаунта"

Переходы:
- "Назад" -> предыдущий экран
- "Выйти из аккаунта" -> `/login`

## MVP-функции
- вход и регистрация как интерфейсные сценарии
- создание проекта
- ввод текста предметной области
- загрузка файла
- выбор типа диаграммы
- генерация результата
- просмотр результата
- сохранение проекта
- список проектов
- редактирование профиля

## Пока не реализуется
- полноценная авторизация с безопасностью production-уровня
- drag-and-drop редактор диаграмм
- роли пользователей
- версии проектов
- экспорт в сложные форматы
